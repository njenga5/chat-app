# Chat App Project
A chatting application using WebRTC as the transport protocol.

## Features
*Private messaging*
*Group messaging*
*Message persistence*
*Showing online status of a user*

## Installing
```sh
git clone https://github.com/njenga5/chat-app.git
cd chat-app
yarn && yarn run electron:serve
```
